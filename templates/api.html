Hello! This is the api endpoint.

/api/album will return the album name in plain text
/api/name will return the song name in plain text
/api/artist will return the artist in plain text
/api/image will return the url for the album art in a png format.
/api/ver returns a value which increases every time the data is refreshed.
/api/refresh manually refreshes the date. Returns false when nothing changed and true when data has been updated.
/api/refresh returns "True" when song is playing, returns "False" when not.
/api/inc Increments /api/ver. Will return number.
